# comedorgatosIntroElectro-Monse Segura
# Descripción

Sistema que dispensa alimento de forma automática para gatos utilizando
Arduino / ESP32 y un mecanismo de motor (servo), Este sistema se implementó en la aplicación Wokwi.
El objetivo de este proyecto es asegurar que el gato reciba porciones controladas de comida
en horarios definidos sin intervención constante de su dueño dueño.

## Funcionamiento

- El motor acciona un mecanismo que libera una porción adecuada de alimento.
- El sistema se puede activar mediante:
  - Un botón manual, y/o
  - Un temporizador programado.
- El microcontrolador controla el tiempo de activación del motor
  para regular la cantidad de comida que se estará dispensando.
